# New Spec

Plan a new product or analyze an existing product and create comprehensive specifications.

## Multi-Agent Mode

This command orchestrates multiple specialist agents to systematically analyze requirements and create detailed product specifications.

### Usage

For new products:
- Analyzes product vision and requirements
- Creates comprehensive product specifications
- Sets up Agent OS in the codebase
- Generates task lists for implementation

For existing products:
- Analyzes current codebase structure
- Identifies areas for enhancement
- Creates upgrade specifications
- Plans migration strategies

### Execution

Refer to the instructions located in this file:
@.agent-os/instructions/core/new-spec.md

### Multi-Agent Coordination

This command uses the following specialist agents:
- **Product Planner**: Analyzes requirements and creates product mission
- **Specification Writer**: Creates detailed technical specifications  
- **Task Creator**: Breaks down specifications into executable tasks
- **Integration Coordinator**: Ensures compatibility with existing systems
- **Quality Assurance**: Validates specification completeness and quality

### Profile Support

Different profiles provide specialized approaches:
- **default**: Standard product specification workflow
- **fullstack**: Full-stack application specification focus
- **frontend**: Frontend-heavy application specialization  
- **backend**: Backend and API-focused specifications

# Validate System

## Overview

Perform comprehensive system-wide validation and health checks across the entire Agent OS installation and project structure. This command validates:
- ✅ Agent OS installation integrity
- ✅ Configuration file correctness
- ✅ Cross-references between commands and instructions
- ✅ Agent instruction file completeness
- ✅ Project structure consistency
- ✅ Development environment setup
- ✅ Dependencies and tooling
- ✅ Git repository health

**Use this when:** Validating Agent OS installation, system health, or troubleshooting issues.

**Use validate-quality for:** Code-specific quality validation.

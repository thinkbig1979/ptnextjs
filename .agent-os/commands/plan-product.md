# Plan Product

## Overview

Plan new products from scratch and install Agent OS framework in new codebases. This command guides you through:
- ✅ Product vision and requirements gathering
- ✅ Agent OS installation and configuration
- ✅ Initial project structure setup
- ✅ Technology stack selection
- ✅ Development standards establishment

**Use this when:** Starting a new product from scratch or installing Agent OS in a new codebase.

**Use analyze-product when:** Working with an existing codebase that needs Agent OS integration.

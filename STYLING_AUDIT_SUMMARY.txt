                        VIOLATION CATEGORIES
================================================================================

1. TEXT COLOR VIOLATIONS (85+ instances)
   - text-gray-* classes (60+): Should use text-muted-foreground or text-foreground
   - text-blue-* classes (25+): Should use text-accent

   Examples:
   ❌ text-gray-600 dark:text-gray-400  →  ✅ text-muted-foreground
   ❌ text-gray-900 dark:text-white     →  ✅ text-foreground
   ❌ text-blue-600 dark:text-blue-400  →  ✅ text-accent

2. BACKGROUND COLOR VIOLATIONS (65+ instances)
   - bg-gray-* classes (45+): Should use bg-muted or bg-background
   - bg-white classes (10+): Should use bg-card or bg-background
   - Progress bars with gray (5+): Should use bg-muted

   Examples:
   ❌ bg-gray-50 dark:bg-gray-900    →  ✅ bg-background
   ❌ bg-gray-100 dark:bg-gray-800   →  ✅ bg-muted
   ❌ bg-white dark:bg-slate-900     →  ✅ bg-card

3. BORDER COLOR VIOLATIONS (20+ instances)
   - border-gray-* classes (15+): Should use border-border
   - border-blue-* classes (5+): Should remove or use badge variants

   Examples:
   ❌ border-gray-200 dark:border-gray-700  →  ✅ border-border
   ❌ border-gray-300                        →  ✅ border-border

4. BADGE/STATUS VIOLATIONS (15+ instances)
   - Hard-coded badge colors instead of shadcn variants
   - Inline color definitions for status states

   Examples:
   ❌ className="bg-blue-100 text-blue-800"  →  ✅ variant="default"
   ❌ className="bg-gray-100 text-gray-800"  →  ✅ variant="outline"

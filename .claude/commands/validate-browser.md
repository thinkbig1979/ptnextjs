# Validate Browser

## Overview

Perform comprehensive browser-based validation and testing for web components, UI implementations, and user-facing features. This command executes:
- ✅ Automated Playwright browser testing
- ✅ Visual regression detection
- ✅ Cross-browser compatibility verification
- ✅ Responsive design validation
- ✅ User interaction flow testing
- ✅ Accessibility compliance checks

**Use this when:** Validating web UI components, user flows, or visual implementations.

**Use validate-quality for:** Code quality, architecture, and non-browser validation.

@.agent-os/instructions/core/validate-browser.md

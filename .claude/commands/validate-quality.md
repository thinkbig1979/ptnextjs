# Validate Quality

## Overview

Perform comprehensive quality validation for specifications, implementations, and codebase standards. This command validates:
- ✅ Specification completeness and clarity
- ✅ Implementation correctness and quality
- ✅ Test coverage and effectiveness
- ✅ Code style and formatting standards
- ✅ Architecture and design patterns
- ✅ Documentation accuracy and completeness
- ✅ Security and error handling
- ✅ Performance considerations

**Use this when:** Validating code quality, architecture, specifications, or implementation completeness.

**Use validate-browser for:** Web UI and browser-based testing.

# Analyze Product

## Overview

Analyze existing codebases to understand their structure, patterns, and requirements before installing Agent OS. This command provides:
- ✅ Comprehensive codebase analysis
- ✅ Technology stack identification
- ✅ Architecture pattern recognition
- ✅ Agent OS installation tailored to existing patterns
- ✅ Integration with existing development workflows

**Use this when:** Installing Agent OS in an existing codebase with established patterns.

**Use plan-product when:** Starting from scratch with a new project.
